# XP-rewiring-
XP rewiring is a one man project focused on enforcing change through dopamine hacks, It's probably my finest work yet
XP Habit Rewiring App 
A single-file HTML app that rewires your brain using an RPG-inspired XP system where rewards only unlock after consistent effort, built for mobile and desktop with zero dependencies.

Core Essence
Philosophy: Effort → XP → Reward (no shortcuts)

Format: One HTML file, runs anywhere

Goal: Build consistency through small daily actions

Design: Mobile-first, dark theme, touch-friendly

What It Does
Earn XP by completing daily tasks (10-25 XP each, 100 XP daily cap)

Fight Bosses on high-resistance tasks (30-50 XP, 1/day)

Level Up with exponential XP thresholds

Unlock Rewards strictly based on XP earned

Daily Check-in required to maintain streak

Works offline - all data saved locally

Key Features
✅ Zero setup - Just open HTML file

✅ Responsive - Phone, tablet, desktop

✅ Offline - No internet needed after first load

✅ No fluff - No social, no badges, no guilt

✅ Private - All data stays on your device

Behavior Design
Default tasks: Focused work, reading, writing, learning

Boss fights: Tackle procrastination head-on

Rewards: Music, browsing, guilt-free breaks

Psychology: Small wins, visible progress, loss aversion

Ship ugly. Fix later. The goal is 30 days of consistency.
